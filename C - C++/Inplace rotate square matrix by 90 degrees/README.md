<h1 align=center>Inplace rotate square matrix by 90 degrees</h1>

## Problem Statement
For a given matrix, rotate it by 90 degrees without using any extra spaces.

### Sample Input

1 2 3<br>
4 5 6<br>
7 8 9<br>

### Sample Output
7 4 1<br>
8 5 2<br>
9 6 3<br>
